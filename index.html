Cómo funciona

En la sección “Actualizar” escribes el texto y pulsas el botón. La página te genera automáticamente el bloque JSON que debes pegar dentro del archivo. Luego haces commit y GitHub Pages publicará la nueva versión.

Consejo práctico: si quieres que sea todavía más fácil, puedes editar solo el archivo notes.json. El resto del diseño no hace falta tocarlo.

Publicarlo en GitHub Pages

  1. Crea un repositorio, por ejemplo nuestro-rincon.
  2. Sube index.html, styles.css y notes.json.
  3. En GitHub, entra en Settings → Pages.
  4. Elige la rama main y la carpeta /root.
  5. Cada día edita notes.json, haz commit y espera la publicación.
